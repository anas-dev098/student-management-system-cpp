# Student Management System (C++)

A simple console-based Student Management System built using C++.
This project helps to understand basic concepts of C++ such as
Arrays,Loops,OOPS,Class.

---

## 📌 Features
- Add student details
- Display all students


---

## 🛠️ Technologies Used
- Language: C++
- Concepts:
  - Class
  - Arrays
  - Functions
  - Loops
  - OOPS

---

## ▶️ How to Run
1. Open terminal in the `src` folder
2. Compile the program:
g++ main.cpp styudent.cpp -o app
  

3. Run the program:
   ./app
